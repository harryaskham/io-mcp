Quick Start

Get running in three commands.

# Install and run
nix develop
uv run io-mcp

# Demo mode (no agent needed)
uv run io-mcp --demo

# Send a message from another terminal
io-mcp-msg "check this file"

Configuration

Global config at ~/.config/io-mcp/config.yml, project overrides in .io-mcp.yml. 

config:
  colorScheme: nord
  tts:
    model: gpt-4o-mini-tts
    speed: 1.3
    emotion: happy
  ambient:
    enabled: true
    initialDelaySecs: 30
  agents:
    hosts:
      - name: Desktop
        host: desktop.local

# Project-local quick actions
quickActions:
  - key: "!"
    label: Commit and push
    action: message
    value: "commit all changes and push"
