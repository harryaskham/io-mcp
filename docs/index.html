How It Works

Three components, one seamless loop.

Scroll & Pick

Claude presents choices. You scroll with a smart ring or your phone. Each option is read aloud through your earphones.

Speak or Type

Give voice commands, have a conversation, or type when you prefer. io-mcp transcribes and sends your input to the agent.

Code Ships

Claude writes code, runs tests, commits. You stay eyes-free the entire time. Narrated progress keeps you in the loop.

Four Ways to Interact

From scroll-and-pick to free conversation — choose the mode that fits the moment.

🎯

Choice Mode

Scroll through options, hear each one read aloud, select with a click or Enter

🗣

Conversation

Continuous voice back-and-forth — like a phone call with your AI coding partner

⌨

Freeform Input

Type replies with live espeak readback so you know what you're writing

📨

Message Queue

Queue messages while the agent works — they'll arrive at its next response

Features

Everything you need for hands-free AI coding.

📡

Multi-Agent Tabs

Run multiple Claude Code instances, each in its own tab with independent state and TTS voice.

🎛

Agent Spawner

Press t to launch new agents locally or on remote hosts via SSH.

📊

Dashboard

Press d for a mission control overview of all active agents and their status.

⚡

Quick Actions

Configurable macros: “commit and push”, “run tests” — one key away.

📜

Agent Log

Press g for a scrollable transcript of everything the agent said and did.

↩

Undo Selection

Press u to go back if you scrolled past the right option.

🔍

Choice Filter

Press / to type and filter choices by name or description.

🎨

Color Schemes

Nord, Tokyo Night, Catppuccin, or Dracula. Switch from the settings menu.

🔔

Audio Cues

Subtle chimes for new choices, selections, agent connections, and recording state changes.

⏱

Ambient Mode

Escalating TTS updates during long silence so you always know what's happening.

📱

Android App

Jetpack Compose companion — touch, scroll, mic, notifications. TTS stays on the phone.

💬

CLI Messaging

io-mcp-msg "check this" — send messages from any terminal pane.

📳

Haptic Feedback

Vibration on scroll and selection. Feels tactile even without seeing the screen.

🛡

Health Monitoring

Automatic detection of stuck or crashed agents with webhook notifications via ntfy, Slack, or Discord.

Keyboard Shortcuts

All configurable via config.keyBindings. Designed for one-handed use.

Quick Start

Up and running in seconds.

1

Install & Run

# Clone and enter dev shell
git clone https://github.com/harryaskham/io-mcp
cd io-mcp && nix develop

# Start io-mcp
uv run io-mcp
2

Connect Claude Code

# In Claude Code, add as MCP server
# io-mcp auto-starts its proxy on :8444
# Agents register and get their own tabs
3

Try Demo Mode

# No agent needed â€” test the scroll UI
uv run io-mcp --demo

# Send a message from another terminal
io-mcp-msg "check this file"

Configuration

Global config at ~/.config/io-mcp/config.yml, project overrides in .io-mcp.yml.

TTS & Voice
config:
  tts:
    model: gpt-4o-mini-tts
    speed: 1.3
    emotion: friendly
    voiceRotation:
      - sage
      - coral
      - vale
Quick Actions
quickActions:
  - key: "!"
    label: Commit and push
    action: message
    value: "commit all changes and push"
  - key: "@"
    label: Run tests
    action: command
    value: "pytest tests/ -q"
Remote Agents
config:
  agents:
    hosts:
      - name: Desktop
        host: desktop.local
        workdir: ~/projects
      - name: Server
        host: gpu-box.tail
        workdir: ~/ml
Notifications
config:
  notifications:
    enabled: true
    channels:
      - name: phone
        type: ntfy
        events:
          - health_warning
          - health_unresponsive

Architecture

Modular design — the proxy survives backend restarts so agents stay connected.

MCP Proxy
:8444

Thin FastMCP proxy. Agents connect here. Survives TUI restarts.

Backend
:8446

Main process with TUI, TTS engine, session management, and tool dispatch.

Frontend API
:8445

REST + SSE for remote frontends. Powers the Android app.

TUI
Textual

Tabbed terminal interface with scroll navigation, TTS, voice input, and themes.
